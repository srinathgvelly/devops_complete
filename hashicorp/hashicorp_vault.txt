Hashicorp Vault : It is designed to help organisation manage access to secrets and transmits them safely with in organization.
				  It makes it very easy to control and manage access by providing you with a unilateral interface to manage every secret in infrastructure.

How Hashicorp Vault Works :	Vault is a secret management tool specifically designed to control access to sensitive credentials in a low trust environmental.
							Vault can be used to authenticate user to make sure they are using authorized to use particular file.
							Hashicorp Configuration Language (HCL) are used to determine who gets the access.
hashicorp Vault used for : 	Secret Management
							Identity based access (Human Permissions)
							Data Encryption
							Secure secret storage.
							
Vault runs on http://localhost:8200
